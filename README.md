# 이루리 강의 영상 다운로더

## 개인정보처리방침 (Privacy Policy)
이루리 강의 영상 다운로더 (이하 "본 확장 프로그램")는 사용자의 개인정보를 중요하게 생각하며, 관련 법령 및 Google의 개발자 프로그램 정책을 준수합니다.

### 1. 개인정보의 수집 및 이용 목적
본 확장 프로그램은 사용자의 어떠한 개인 식별 정보(이름, 이메일, 전화번호 등)도 수집, 저장, 공유 또는 외부 서버로 전송하지 않습니다. 모든 동작은 사용자의 브라우저 로컬 환경에서만 수행됩니다.

### 2. 수집하는 데이터 항목 및 기술적 활용
본 확장 프로그램은 서비스 제공을 위해 최소한의 기술적 데이터만 활용하며, 그 내역은 다음과 같습니다.

웹사이트 콘텐츠: 사용자가 접속한 강원대학교 이루리(LXP) 시스템 내 강의 페이지에서 영상 스트리밍 주소(M3U8/MP4)를 추출하기 위해 사용됩니다.

기기 정보 및 로그: 어떠한 기기 정보나 사용 기록도 수집하지 않습니다.

### 3. 권한 사용에 대한 정당성
Google 정책에 따라 본 프로그램이 요청하는 권한의 사용 목적을 명시합니다.

storage: 사용자의 다운로드 대기열 목록과 약관 동의 여부를 브라우저 로컬에 저장하기 위해 사용합니다.

activeTab: 현재 활성화된 강의 탭에서 영상 소스 URL을 안전하게 감지하고 추출하기 위해 사용합니다.

downloads: 병합이 완료된 영상 파일을 사용자의 기기에 파일 형태로 저장하기 위해 사용합니다.

offscreen: 대용량 영상 병합 작업 시 브라우저 성능을 유지하기 위해 백그라운드 환경을 생성하는 목적으로만 사용합니다.

### 4. 데이터 보안 및 전송
본 프로그램은 **원격 코드(Remote Code)**를 실행하지 않으며, 모든 로직은 패키지 내부의 스크립트로만 동작합니다.

추출된 영상 데이터 및 모든 처리 과정은 사용자의 컴퓨터 내부(Origin Private File System)에서만 이루어지며 제3자에게 전송되지 않습니다.

### 5. 사용자의 책임 및 저작권 고지
본 프로그램은 개인적인 학습 및 복습을 위한 학습 보조 도구입니다.

다운로드한 콘텐츠를 무단으로 배포, 공유 또는 상업적으로 이용하는 행위는 저작권법에 위배될 수 있으며, 이로 인해 발생하는 모든 법적 책임은 사용자 본인에게 있습니다.

본 프로그램은 기술적 보호조치(DRM) 무력화 기능을 포함하고 있지 않습니다.
